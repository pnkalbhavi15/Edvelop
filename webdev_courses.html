HTML (Hyper Text Markup Language) is the standard markup language for creating Web pages.It tells the tell the browser how to display the content.

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

JavaScript is a language that allows us to implement complex features on web pages like displaying timely content updates, interactive maps,etc.

Typescript is a strict syntactical superset of JavaScript and adds optional static typing to the language. It is designed for the development of large applications.
